# roamlii-dashboard

---

## Installation & Setup

1. Clone the repository:  
   ```bash
   git clone https://github.com/thee24/roamlii-dashboard.git
   cd roamlii-dashboard
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the project:

   ```bash
   npm run build
   ```

4. Start the project:

   ```bash
   npm run start
   ```

---

## Usage

Open your browser and navigate to `http://localhost:3000` to access the dashboard.
